# Insurance Data Analysis Project

This project simulates insurance policy sales and claims data and performs analytical queries to evaluate profitability and risk.

## Tools Used
- Python
- Pandas
- Jupyter Notebook


## Dataset
1. Policy Sales Data
2. Claims Data

## Analysis Performed
- Total premium collected
- Claim cost analysis
- Claim to premium ratio
- Loss ratio calculation
- Future claim liability estimation
